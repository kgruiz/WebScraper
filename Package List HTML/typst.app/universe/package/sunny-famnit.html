Thesis template for University of Primorska, FAMNIT

Create project in app

University of Primorska,

Faculty of Mathematics, Natural Sciences and Information Technologies

This is a Typst template for FAMNIT final work.

Abbreviations (kratice)

You can specify Abbreviations at the start as an attribute kratice and pass it a dictionary of the abbriviation and itâs explanation. Then you can reference them in text using @<short name> to create a link to it.

Attachments

Some thesis need Attachments that are shown at the end of the file. To add these attachments add them in your project under priloge as a dictionary of the attachment name and its content. I suggest having a seperate attachments.typ file, from where you can reference them in the main project.

Language

The writing of the thesis can be achieved in two languages; Slovene and English. They have some differences between them in the way the template is generated, as the thesis needs to be different for each one. you can specify the language with the text_lang attribute.

How to use

Click the button above to create a new project using this template in the Typst app.

You can also use the Typst CLI to start a new project on your computer using this command: 

typst init @preview/sunny-famnit:0.2.0
