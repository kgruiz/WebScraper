A basic resume template in typst.

To compile/watch, make sure you pass the argument --font-path ./fonts/ to typst-cli.

If you use the helix editor, a configuration has been given in the repository. This auto-compiles your document on saving and you can preview the results in real time using a pdf viewer. Also provides autocomplete, code renaming, and other cool (LSP) features.

Make sure you have tinymist and typstyle installed before using the helix config.
