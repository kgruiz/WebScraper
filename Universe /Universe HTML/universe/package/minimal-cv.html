Usage

From Typst app

Create a new project based on the template minimal-cv .

Locally

Copy the template to your Typst project.

From a blank project

Import the library : 

#import "@preview/minimal-cv:0.1.0": *

Show the root cv function : 

#show: cv.with(
  theme: (),
  title: "YOUR NAME",
  subtitle: "YOUR POSITION",
  aside: [
    ASIDE CONTENT
  ]

MAIN CONTENT

Several content functions are available.

Section 

#section(
  theme: (),
  "TITLE_CONTENT",
  "BODY_CONTENT",
)

Entry 

#entry(
  theme: (),
  right: "FLOATING_CONTENT",

  "GUTTER_CONTENT",
  "TITLE_CONTENT",
  "BODY_CONTENT",
)

Progress bar 

#progress-bar(50%)

How to use

Click the button above to create a new project using this template in the Typst app.

You can also use the Typst CLI to start a new project on your computer using this command: 

typst init @preview/minimal-cv:0.1.0
