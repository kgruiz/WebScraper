This template exports the \texttt{\ report\ } function with the
following named arguments:

\begin{itemize}
\tightlist
\item
  \texttt{\ title\ } : The reoirtâ€™s title as string. This is displayed

\item
  \texttt{\ subtitle\ } : The reportâ€™s subtitle as string. This is

\item
