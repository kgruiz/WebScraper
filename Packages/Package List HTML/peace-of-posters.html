peace-of-posters (PoP) is a Typst package to help creating scientific posters. It is flexible and can be used for different sizes and layouts. To see what is possible have a look at some of my own real-world examples in the showcase section of the documentation.

Documentation

The external documentation is coming along slowly. Most notably, there are examples and showcases missing but I hope to be adding them over the coming months.
