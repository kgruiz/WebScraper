IU thesis template in Typst

Simple template which meets thesis and dissertation requirements of IU Bloomingtonâs Graduate School. This template provides a couple functions: 

#show: thesis.with(
    title: [content],
    author: [content],
    dept: [content],
    year: [content],
    month: [content],
    day: [content],
    committee: (
        (
            name: "Person 1",
            title: "Ph.D.",
        ),
    ),
    dedication: [content],
    acknowledgement: [content],
    abstract: [content],
)

and 

#iuquote(content)

Everything else follows along with basic Typst syntax such as headings, figures, tables, etc. See the provided template as an example.

How to use

Click the button above to create a new project using this template in the Typst app.

You can also use the Typst CLI to start a new project on your computer using this command: 

typst init @preview/modern-iu-thesis:0.1.0
