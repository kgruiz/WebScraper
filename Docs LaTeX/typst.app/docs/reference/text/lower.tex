\section{\texorpdfstring{\texttt{\ lower\ }}{ lower }}\label{summary}

Converts a string or content to lowercase.

\subsection{Example}\label{example}

\begin{verbatim}
#lower("ABC") \
#lower[*My Text*] \
#lower[already low]
\end{verbatim}
