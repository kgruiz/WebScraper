\begin{verbatim}
$ a'''_b = a^'''_b $
\end{verbatim}

\subsection{Syntax}\label{syntax}

This function has dedicated syntax: use apostrophes instead of primes.
They will automatically attach to the previous element, moving
superscripts to the next level.
